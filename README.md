Simple3DViewer — 3D-рендерер для просмотра и визуализации 3D-моделей в формате OBJ.

## О проекте

В проекте реализованы:
- загрузка 3D-моделей из OBJ-файлов
- отображение модели в графическом интерфейсе JavaFX
- работа с 3D-геометрией и матрицами преобразований
- базовые алгоритмы растеризации и z-буфер для отображения поверхности
- удаление вершин и полигонов из модели
- чтение и запись OBJ-файлов

## Как запустить

1. Откройте проект в среде разработки, поддерживающей Maven.
2. Убедитесь, что в системе установлен JDK 17 или совместимый.
3. Соберите проект командой:
   ```bash
   mvn clean package
   ```
4. Запустите приложение через класс `com.cgvsu.Main`.

## Структура проекта

- `src/main/java/com/cgvsu` — основной код приложения
- `src/main/resources/com/cgvsu/fxml` — интерфейс JavaFX и стиль
- `src/main/java/com/cgvsu/objreader` — чтение OBJ-файлов
- `src/main/java/com/cgvsu/objwriter` — запись OBJ-файлов
- `src/main/java/com/cgvsu/Math` — математика для векторов и матриц
- `src/main/java/com/cgvsu/render_engine` — камера, сцена и z-буфер
- `src/main/java/com/cgvsu/Rasterization` — компоненты треугольников и растеризации
